# Souvik Ghosh — Portfolio

A responsive, single-page portfolio website showcasing projects, experience, certifications, and contact information.

## Sections

- **Hero** — Introduction and key highlights
- **About** — Bio and stats (stability, performance, API response, uptime)
- **Tech Stack** — Technologies and tools (Java, Python, AWS, Docker, etc.)
- **Projects** — Banking Web App (finance dashboard), Automated Email Sender
- **Experience** — J.P. Morgan Software Engineering Virtual Experience
- **Certifications** — SQL (Advanced), Problem Solving (HackerRank), AWS Cloud Practitioner
- **Contact** — Email, LinkedIn, GitHub links and contact form (mailto)

## Tech

- **HTML5** — Semantic structure, accessibility (skip link, ARIA)
- **CSS3** — Custom properties, Grid/Flexbox, responsive design, scroll reveal
- **JavaScript** — Navbar scroll effect, mobile menu, smooth scroll, form mailto, scroll reveal

## Run locally

1. Clone the repo:
   ```bash
   git clone https://github.com/xouvik09/porfolio.git
   cd porfolio
   ```
2. Open `index.html` in a browser, or serve the folder (e.g. `npx serve .` or VS Code Live Server).
